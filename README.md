# Savy Nehemiah Checkout

This is a Next.js project initialized with a responsive payment checkout UI.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

2. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- Next.js 14
- React
- Tailwind CSS
- Lucide React (Icons)
